core/httpErrors~HTTP_ERRORS

The list of HTTP errors with, for each error, its associated hexadecimal code and HTTP return code. HTTP errors are sent by {{#crossLinkModule "controllers"}}{{/crossLinkModule}}.

var httpErrors = process.require('app/server/httpErrors.js');
